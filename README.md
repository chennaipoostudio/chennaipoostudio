## ğŸš€ Deploying to GitHub Pages

Follow these steps to deploy your website to GitHub Pages:

### Step 1: Initialize Git Repository

Open Command Prompt or PowerShell in the project folder and run:

```bash
cd "C:\Users\elumalai.p2\Desktop\chennai-poo-studio"
git init
git add .
git commit -m "Initial commit: Chennai Poo Studio website"
```

### Step 2: Create GitHub Repository

1. Go to [GitHub](https://github.com) and log in
2. Click the "+" icon in the top right corner
3. Select "New repository"
4. Name it: `chennai-poo-studio` or `your-username.github.io` (for personal site)
5. Keep it **Public**
6. **Do NOT** initialize with README, .gitignore, or license
7. Click "Create repository"

### Step 3: Connect and Push to GitHub

Copy the commands from GitHub (they'll look similar to below) and run them:

```bash
git remote add origin https://github.com/your-username/chennai-poo-studio.git
git branch -M main
git push -u origin main
```

**Alternative using GitHub CLI** (if you have `gh` installed):

```bash
gh repo create chennai-poo-studio --public --source=. --remote=origin --push
```

### Step 4: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click on **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under "Source", select:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Click **Save**
6. Wait a few minutes for deployment

### Step 5: Access Your Website

Your website will be live at:
- `https://your-username.github.io/chennai-poo-studio/`

Or if you named it `your-username.github.io`:
- `https://your-username.github.io/`
